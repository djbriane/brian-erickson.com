# OpenAI Codex

Follow [AGENTS.md](AGENTS.md) for repository conventions.

## Git

| Item | Convention |
|------|------------|
| Branch | `feature/<short-kebab-description>` |
| Commit | `type(scope): imperative summary` |
| PR title | Same conventional format |
| Signing | GPG-signed commits expected |

## Types & Scopes

Types: `feat`, `fix`, `docs`, `refactor`, `test`, `chore`, `perf`, `build`, `ci`

Suggested scopes: `content`, `blog`, `projects`, `resume`, `seo`, `styles`, `spec`, `specs`, `deps`, `ci`

## Before Finishing

1. Read [CONTEXT.md](CONTEXT.md) and the current build-out spec.
2. Run relevant tests and `npm run build` once the project scaffold exists.
3. Do not commit or open PRs unless the user explicitly asks.

Full guide: [CONTRIBUTING.md](CONTRIBUTING.md)
